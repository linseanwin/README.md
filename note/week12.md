# BFS

![](https://qph.fs.quoracdn.net/main-qimg-7b145b86b4bca7d619028bc160f8332a)

廣度優先搜尋法，是一種圖形搜索演算法。從圖的某一節點開始走訪，接著走訪此一節點所有相鄰且未拜訪過的節點，由走訪過的節點繼續進行先廣後深的搜尋。以樹來說即把同一深度的節點走訪完，再繼續向下一個深度搜尋，直到找到目的節點或遍尋全部節點。
